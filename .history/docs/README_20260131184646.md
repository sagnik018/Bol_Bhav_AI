# Bol Bhav AI

Bol Bhav AI is a real-time, voice-first negotiation assistance platform for local market vendors.
It helps vendors make informed pricing decisions using voice input, market price intelligence,
and simple AI-driven negotiation guidance.

## Problem
Local vendors often suffer losses due to lack of real-time market price awareness,
language barriers, and poor negotiation confidence.

## Solution
Bol Bhav AI enables vendors to:
- Speak instead of type
- Get live market price reference
- Receive clear negotiation guidance (Hold / Counter / Accept / Walk Away)

## Tech Stack
Frontend: React, Web Speech API, CSS  
Backend: Node.js, Express  
Database: MongoDB  
AI Logic: Rule-based (MVP)
